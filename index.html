Privacy Policy for PerseoAI Meeting Assistant

**Last Updated: December 11, 2024**

Introduction

PerseoAI ("we," "our," or "us") operates the PerseoAI Meeting Assistant browser extension. This Privacy Policy explains how we collect, use, disclose, and safeguard your information when you use our extension.

**By using PerseoAI Meeting Assistant, you agree to the collection and use of information in accordance with this policy.**

Information We Collect

1. Meeting Transcripts

  • Audio transcriptions from Google Meet calls
  • Speaker identification data
  • Meeting timestamps and duration
  • Participant information (names, if available)

2. User Account Information

  • Email address (for authentication)
  • User preferences and settings
  • Extension configuration data

3. Usage Data

  • Extension usage statistics (anonymous)
  • Feature usage patterns
  • Error reports and crash logs
  • Performance metrics

4. Technical Data

  • Browser type and version
  • Operating system
  • Extension version
  • IP address (for security purposes)

How We Use Your Information

We use the collected information for:

1. **Providing Services**

- Real-time transcription of meetings

- Generating AI-powered summaries

- Storing and organizing meeting history

- Exporting transcripts

2. **Improving Our Services**

- Enhancing transcription accuracy

- Developing new features

- Fixing bugs and improving performance

- Understanding user needs

3. **Security & Authentication**

- Verifying user identity

- Preventing fraud and abuse

- Protecting against security threats

4. **Communication**

- Responding to support requests

- Sending important updates

- Notifying about new features (with consent)

Data Storage & Security

Storage Location

  • Transcripts are stored in encrypted Supabase databases
  • User preferences stored locally in browser storage
  • Cloud backup (optional, requires explicit consent)

Security Measures

  • **Encryption**: All data encrypted in transit (HTTPS/TLS) and at rest
  • **Access Control**: Strict access controls and authentication
  • **Regular Audits**: Security audits and vulnerability assessments
  • **Secure Infrastructure**: Enterprise-grade hosting on Supabase

Data Retention

  • **Meeting Transcripts**: Retained until you delete them
  • **Usage Data**: Aggregated data retained for 90 days
  • **Account Data**: Retained until account deletion
  • **Deleted Data**: Permanently removed within 30 days

Third-Party Services

We use the following third-party services:

1. **Supabase** (Database & Authentication)

- Stores encrypted meeting transcripts

- Manages user authentication

- Privacy Policy: https://supabase.com/privacy

2. **Google Gemini AI** (AI Processing)

- Generates meeting summaries

- Extracts action items

- Privacy Policy: https://policies.google.com/privacy

3. **Deepgram/AssemblyAI** (Transcription)

- Converts speech to text

- Speaker identification

- Privacy Policy: https://deepgram.com/privacy / https://www.assemblyai.com/legal/privacy-policy

**Important**: These services process meeting audio/transcripts to provide functionality. They do not store or use your data for training their models without consent.
